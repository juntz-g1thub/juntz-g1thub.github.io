# 欢迎访问我的新博客

发表时间：2026-03-06

欢迎来到我的新博客！这是我的第一篇博客文章，介绍这个零构建博客项目。

## 为什么选择零构建？

之前的博客使用了Eleventy（一个静态站点生成器），但我觉得太复杂了。所以我决定回到基础：

### 核心设计理念

1. **简单** - 无需构建步骤
2. **快速** - 浏览器直接渲染Markdown
3. **透明** - 所有代码都是纯HTML/CSS/JS
4. **灵活** - 完全自定义界面样式

## 技术栈

这个博客使用了最简单的技术栈：

- **HTML5** - 页面结构
- **CSS3** - 样式设计（带暗色模式）
- **JavaScript** - 核心逻辑（约200行代码）
- **marked.js** - Markdown渲染（CDN）

无需Webpack、Babel、NPM，甚至不需要Node.js！

## 工作原理

1. 浏览器加载一个HTML页面
2. JavaScript加载Markdown文件
3. 使用marked.js将Markdown转为HTML
4. 更新页面内容

整个过程完全在浏览器中完成。
